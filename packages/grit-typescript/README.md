# grit-typescript

> Create a brand new typscript project

## Usage

Install [Grit](https://github.com/TimCrooker/grit) first.

```bash
yarn global add grit-cli
# or
npm i -g grit-cli
```

### From npm

```bash
grit run typescript my-project
```

### From git

```bash
grit run TimCrooker/grit-typescript my-project
```
