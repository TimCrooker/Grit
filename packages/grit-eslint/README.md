# grit-eslint

> my awesome new project

## Usage

Install [Grit](https://github.com/TimCrooker/projenerator) first.

```bash
yarn global add grit-cli
# or
npm i -g grit-cli
```

### From npm

```bash
grit eslint my-project
```

### From git

```bash
grit timcrooker/grit-eslint my-project
```
