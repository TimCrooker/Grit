# grit-readme

> my awesome new Grit generator

## Usage

Install [Grit](https://github.com/TimCrooker/projenerator) first.

```bash
yarn global add grit-cli
# or
npm i -g grit-cli
```

### From npm

```bash
grit readme my-project
```

### From git

```bash
grit TimCrooker/grit-readme my-project
```
