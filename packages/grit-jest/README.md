# grit-jest

> Add jest testing to any node project

## Usage

Install [Grit](https://github.com/TimCrooker/projenerator) first.

```bash
yarn global add grit-cli
# or
npm i -g grit-cli
```

### From npm

```bash
grit jest my-project
```

### From git

```bash
grit timcrooker/grit-jest my-project
```
